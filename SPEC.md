| 項目 | 内容 |

| 言語 | Python |

---

## 2. 主要機能

### 2-1. 急騰銘柄検知・通知
- 場中リアルタイム監視（数分ごとにスキャン）
- 条件に合致した銘柄をLINEに即時通知
- 急騰条件は後で詳細を決定

### 2-2. チャート分析
- 監視銘柄のインタラクティブチャートを表示（Plotly）
- 足種の切り替えが可能（下記「足種仕様」参照）
- テクニカル指標のオーバーレイ表示
  - 移動平均線（SMA/EMA）
  - ボリンジャーバンド
  - RSI
  - MACD
  - ボリューム

### 2-3. 監視銘柄管理
- 銘柄コードの登録・削除
- 銘柄ごとの通知条件設定

### 2-4. LINE通知
- LINE Messaging API を使用
- 急騰条件合致時の即時通知

## 3. 足種仕様

### 対応足種

| 足種 | yfinance（暫定） | auカブコム証券API（本番） |
|------|----------------|------------------------|
| 秒足 | **15分足で代替表示** | ○ |
| 1分足 | **15分足で代替表示** | ○ |
| 5分足 | **15分足で代替表示** | ○ |
| 10分足 | **15分足で代替表示** | ○ |
| 15分足 | ○ | ○ |
| 日足 | ○ | ○ |
| 週足 | ○ | ○ |
| 月足 | ○ | ○ |

### 実装方針
- UI上は全足種のボタンを表示する
- yfinance使用時、15分足未満が選択された場合は「15分足で表示中（データソースの制限）」と注記を表示する
- データソース切り替えはconfig.pyで管理し、auカブコムAPI対応後は自動的に全足種が有効になる

### yfinanceの足種マッピング

| 選択足種 | yfinanceのinterval指定 |
|---------|----------------------|
| 秒足〜10分足 | `15m`（代替） |
| 15分足 | `15m` |
| 日足 | `1d` |
| 週足 | `1wk` |
| 月足 | `1mo` |

## 9. 急騰検知条件

### 発火ルール
```
（価格条件①）AND（出来高条件②）→ LINE即時通知
テクニカル条件③が重なれば通知メッセージに付加情報として表示
```

### デフォルト閾値（UI上で手動変更可能）

**① 価格条件（いずれか1つ以上）**
| 条件 | デフォルト値 |
|------|------------|
| 前日終値比 | +3%以上 |
| 当日始値比 | +5%以上 |

**② 出来高条件**
| 条件 | デフォルト値 |
|------|------------|
| 過去20日平均出来高比 | 3倍以上 |

**③ テクニカル指標シグナル（付加情報）**
| 条件 | デフォルト値 |
|------|------------|
| RSI | 70以上（過熱） |
| ゴールデンクロス | 短期MA > 長期MAに転換 |
| ボリンジャーバンド突破 | +2σ超え |

### 閾値の管理方針
- **手動設定**: UI上のスライダー・入力欄で銘柄ごとに変更可能
- **将来実装（AI自動設定）**: 銘柄ごとの過去実績データ＋市場ニュースを分析し、最適な閾値をAIが自動提案・設定する機能を追加予定
